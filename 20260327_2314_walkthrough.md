## 改造前后架构对比

```
改造前:
  用户输入 → Flask → if/else 关键词匹配 → 字符串模板填充 → 输出
              ↓
         FFprobe 数据 → 模板变量替换
         Claude（可选, 20s超时, 很少成功）

改造后:
  用户输入 → Flask → 启动 Claude Code Agent Session (300s)
                       ↓
                  Agent 读取 SKILL.md (行为指南)
                  Agent 读取用户需求 + 结构化 brief
                  Agent 调用 probe_source.py (FFprobe)
                  Agent 调用 visual_review.py (Gemini/GLM-4V)
                  Agent 读取 platform-specs.md 等参考资料
                  Agent 自主撰写验收标准 / 质检报告
                       ↓
                  产物写入项目目录 → Flask 展示
```
